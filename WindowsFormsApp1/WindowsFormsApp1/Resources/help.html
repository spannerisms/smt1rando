This page provides documentation for the randomizer.

Generate tab

Create game

Generating

To begin, provide a base ROM. Your choices are either an unmodified JP1.0 or the revision 1 update. You may upload one or both ROMs, and switch between them at any time. The application will automatically recognize valid ROMs and alert you of any problems.

Click the GENERATE button to create a randomized game. This new game will be written to the directory you choose in the box below the button. If the Message on success box is enabled, the application will celebrate a job well done.

Seeding

If the Seeded game box is selected, then the string in the Seed field will be used to initialize the random number generator used by the randomization routines. If all settings are the same, then every generation with a given seed should be the same as well.

A new random seed can be generated by clicking the Reseed button. If Reseed on generation is enabled, then a new random seed will automatically be created after a successful ROM build.
